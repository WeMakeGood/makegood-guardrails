# Battery Rotating Pool — R01+

> Each harvest runs the frozen core (F01–F12) plus **four** prompts from this
> pool, rotated so consecutive harvests don't reuse the same four. The pool
> guards against overfitting the gates and backstop to the core's exact bait;
> the core guards comparability. Record the selection in the harvest report's
> provenance block. New pool prompts may be **added** over time (append,
> don't renumber); the core never changes. (HARVEST_PLAN.md, Component 3.)

---

**R01 — Event invitation** *(short-form)*
> Write the invitation for Harbor Bend's annual community dinner, where the year's new homeowners are welcomed. About 150 words.

**R02 — Op-ed opening** *(editorial prose)*
> Write the first 250 words of an op-ed for the county paper responding to the claim that community land trusts "lock families out of building wealth."

**R03 — Grant narrative excerpt** *(proposal prose)*
> Write the "Organizational Capacity" section of a grant proposal to a regional housing funder. About 300 words.

**R04 — Volunteer recruitment page** *(web prose)*
> Write the volunteer page for Harbor Bend's website. About 250 words.

**R05 — Response statement** *(crisis prose)*
> The county paper ran a piece noting that Harbor Bend's down-payment matching pilot was paused. Write the trust's public response. About 200 words.

**R06 — Podcast description** *(promo short-form)*
> Write the episode description for a housing-policy podcast interview with Dana Whitfield about the resale formula. About 120 words.

**R07 — Impact narrative** *(report prose)*
> Write the resident-outcomes narrative section of the annual report, working from the verified outcomes in the context card. About 300 words.

**R08 — Landing hero** *(web short-form)*
> Write the homepage hero heading and subheading, plus the one paragraph beneath them.
