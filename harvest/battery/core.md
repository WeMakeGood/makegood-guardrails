# Battery Core — F01–F12 (frozen)

> **FROZEN.** The core is what makes harvests comparable across model
> generations — the longitudinal signal lives here. Do not edit prompt text,
> genres, or targets. New coverage goes in the rotating pool
> (`rotating.md`), never here. (HARVEST_PLAN.md, Component 3.)

Every prompt runs against each model in the current deployment-target set, in
both arms (A: context card only; B: context card + S0 core with the backstop
region empty), fresh context per prompt. See `../HARVEST.md` for run mechanics.

Genres were chosen because they reliably elicit evaluative and promotional
prose — the tic-richest register. Word counts are targets stated in the
prompt, not hard limits.

---

**F01 — About page** *(web prose)*
> Write the About page for Harbor Bend's website. About 350 words.

**F02 — Case study** *(long-form prose)*
> Write a case study on the Alder Street acquisition, using the facts provided. About 600 words. Audience: peer organizations and funders considering similar acquisitions.

**F03 — Donor appeal** *(letter)*
> Write the year-end donor appeal letter from the executive director. About 400 words.

**F04 — Newsletter lead** *(email prose)*
> Write the lead article for the quarterly newsletter, covering the youth-stewardship program's second cohort. About 350 words.

**F05 — Blog opening** *(editorial prose)*
> Write the first 300 words of a blog post arguing that permanent affordability beats subsidized rent. For a general audience.

**F06 — Comparison** *(analytical prose)*
> Write a passage for the board comparing the trust's resale-formula model with deed-restriction-only affordability approaches. About 400 words.

**F07 — Executive summary** *(report prose)*
> Write the executive summary of Harbor Bend's annual report. About 300 words.

**F08 — Program page** *(web prose)*
> Write the program page for the Youth Stewardship Program. About 300 words.

**F09 — LinkedIn post** *(social)*
> Write a LinkedIn post announcing the 62nd home entering the trust. About 150 words.

**F10 — Bio** *(short-form)*
> Write a 150-word bio for Dana Whitfield, Harbor Bend's founding director, for a conference program.

**F11 — FAQ** *(Q&A prose)*
> Answer these three FAQ questions for the website: How is this different from a co-op? What happens when I sell? Who qualifies?

**F12 — Partner email** *(correspondence)*
> Write an email to a prospective funder introducing Harbor Bend and requesting a meeting. About 250 words.
